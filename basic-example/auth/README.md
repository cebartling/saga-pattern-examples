# Authentication and authorization service

## Local development

### Virtual Machine options

```
-Xms256m -Xmx512m
```

### Environment variables
- `JDBC_URL=jdbc:postgresql://localhost:15432/auth`
- `JDBC_USERNAME=auth`
- `JDBC_PASSWORD=auth`
